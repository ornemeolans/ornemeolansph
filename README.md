# Ornella Meolans | Fotografía Profesional

Este es el repositorio del sitio web oficial de **Ornella Meolans Fotografía**, un proyecto personal que combina mis dos pasiones: la fotografía y el desarrollo web. El sitio funciona como un portafolio profesional para exhibir mis trabajos, servicios y cursos en la ciudad de Córdoba, Argentina.

## 📸 Sobre el Proyecto
El objetivo principal de esta web es proporcionar una plataforma centralizada donde los clientes puedan explorar mis diferentes servicios fotográficos, conocer mi historia y contactarme para reservas.

### Secciones principales:
- **Inicio**: Galería destacada con acceso directo a las categorías de sesiones.
- **Cursos**: Información sobre formación fotográfica (Nivel Básico, Avanzado y Fotografía con Celular).
- **Sesiones**: Detalle de servicios (Marca Personal, Producto, Embarazadas, Books Personales, etc.).
- **Mi Historia**: Mi trayectoria formativa en la UPC, misión, valores y experiencia profesional.
- **Contacto**: Formulario y enlaces directos a redes sociales.

## 🛠️ Tecnologías Utilizadas
Para el desarrollo de este portafolio utilicé un stack de tecnologías frontend enfocado en el rendimiento y la estética visual:

* **HTML5 & CSS3**: Estructura semántica y estilos personalizados.
* **Bootstrap 5**: Framework para garantizar un diseño responsivo y componentes modernos.
* **JavaScript**: Para funcionalidades interactivas.
* **AOS (Animate On Scroll)**: Biblioteca para animaciones de entrada al hacer scroll.
* **PHP**: Scripting para la gestión del envío de formularios de contacto.
* **Adobe Typekit**: Tipografías personalizadas para mantener la identidad visual.

## 🚀 Características
- **Diseño Responsive**: Adaptado para una navegación fluida en dispositivos móviles, tablets y escritorio.
- **Optimización SEO**: Implementación de meta-tags, descripciones y palabras clave para mejorar el posicionamiento en buscadores.
- **Identidad Visual**: Integración de logotipos y paleta de colores coherente con la marca personal.
